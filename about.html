À propos du projet 📖

Ce projet a été réalisé dans le cadre du TP N°2 sur les workflows Git collaboratifs. Il met en pratique la gestion de branches, les Pull Requests et la résolution de conflits.

Développé par deux développeurs travaillant en parallèle sur des branches feature/ indépendantes, puis intégrées dans main via des Pull Requests documentées et approuvées.
